# 🏬 Supermarket Sales & Operations Analytics

A data analytics project focused on processing retail transaction streams to uncover actionable trends in operational efficiency, customer demographics, and product category margins. This project utilizes Python to perform exploratory data analysis (EDA), engineer time-based features, and visualize revenue drivers.

## 📊 Core Business Insights Uncovered
* **Staff Optimization Windows:** Transaction volume peaks drastically between **1:00 PM – 3:00 PM** and **7:00 PM – 8:00 PM** daily, highlighting key opportunities for optimized cashier staffing schedules.
* **Customer Segment Behavior:** Loyalty members dominate spending within the *Home and Lifestyle* product line by a notable margin, while casual walk-in customers match them evenly in *Food and Beverages*.
* **Branch Performance:** The Yangon branch generates the highest overall revenue, while Mandalay tracks a higher Average Transaction Value (ATV) per basket.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Seaborn, Matplotlib
